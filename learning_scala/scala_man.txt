0.1 sbt

 use sbt for building, testing and running

 file structure:

 src/
    main/
        resources/
        scala/
        java/
    test/
        resources/
        scala/
        java/

 sbt - starts sbt

 console - starts Scala interpreter inside sbt (repl),
  which also loads code from project

 compile - to compile code in src/main/scala

 test - to run tests in src/tests/scala
